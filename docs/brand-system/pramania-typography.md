# Pramania Typography

## Brand direction
Pramania should feel premium, intimate, calm, and editorial.

## Recommended fonts

### Brand serif
Use a refined high contrast serif for brand moments.
Recommended:
- Cormorant Garamond
- EB Garamond
- Canela, if licensed
- Noe Display, if licensed

### UI sans
Use a quiet, clear sans serif for product UI.
Recommended:
- Inter
- Avenir
- Neue Haas Grotesk, if licensed

### Optional mono
- IBM Plex Mono

## Approved usage
- Use the serif for the logo, hero headings, and brand-led screens
- Use the sans for interface, forms, tables, labels, buttons, and body text

## Approved tagline
Career clarity, guided by intelligence
